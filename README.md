# Book & Research Discovery Control Panel

## Group Details
- **Group Size**: 2 Students
- **Member 1 (Leader)**: Jaineel Kantesaria
- **Member 2**: Bhavnesh Soni
## Project Scope
A research and reading web application built using Python, Flask, and SQLite. Users can search for books, authors, or research topics using free APIs and maintain a personalized control panel to track reading statuses.

## External REST APIs Plan
1. **Open Library Search API** - For fetching book titles, authors, and descriptions.
2. **Open Library Covers API** - To render book cover images dynamically.
3. **OpenAlex API** - For pulling research metadata and scientific papers.
